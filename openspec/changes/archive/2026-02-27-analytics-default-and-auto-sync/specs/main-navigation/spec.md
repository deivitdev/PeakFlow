## ADDED Requirements

### Requirement: Default Entry Point
The system SHALL set the "Analytics" screen as the default entry point upon application launch.

#### Scenario: App launch home screen
- **WHEN** the app is opened
- **THEN** the "Analytics" screen is displayed.

### Requirement: Navigation Tab Order
The bottom navigation bar SHALL display tabs in the following order: Analytics, Workouts, Profile.

#### Scenario: Bottom bar display
- **WHEN** the bottom navigation bar is rendered
- **THEN** "Analytics" is the first item on the left.
