## ADDED Requirements

### Requirement: User Profile Data
The system SHALL allow users to store and manage their personal athlete data, including Name, Weight (kg), and Height (cm).

#### Scenario: User saves profile data
- **WHEN** the user enters their name, weight, and height in the profile screen and clicks "Save"
- **THEN** the system SHALL persist this data in local storage

### Requirement: Profile Tab Accessibility
The system SHALL provide a dedicated tab in the bottom navigation bar to access the user profile.

#### Scenario: User navigates to profile
- **WHEN** the user clicks the "Profile" icon in the navigation bar
- **THEN** the system SHALL display the Profile screen
