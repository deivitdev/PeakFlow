## MODIFIED Requirements

### Requirement: Cycling-Specific Layout
The system SHALL provide a layout optimized for cycling that prioritizes power (W), cadence (RPM), speed (KM/H), Aerobic Efficiency, and **Relative Power (W/kg)**.

#### Scenario: View cycling activity
- **WHEN** the user opens a cycling activity with power data
- **THEN** the system SHALL display the `Average W/kg` and `Max W/kg` alongside other power metrics.
