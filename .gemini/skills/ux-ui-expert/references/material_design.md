# Material Design (M3) Key Principles

*   **Platform**: Primarily Android, widely used on Web.
*   **Core Concept**: "Material" as a metaphor. Interfaces are built of paper and ink, with shadows conveying depth and hierarchy.
*   **M3 Additions**: Personalization (Dynamic Color), adaptive layouts, and updated component states.

## Typography
*   Uses a standardized scale (Display, Headline, Title, Body, Label).
*   Default font: Roboto (or system default).

## Color
*   **Dynamic Color**: Extracts colors from the user's wallpaper to create a personalized theme.
*   **Tonal Palettes**: Primary, Secondary, Tertiary, Error, Background, Surface. Each has corresponding "On-" colors for contrast (e.g., `onPrimary` text over `primary` background).

## Elevation & Depth
*   Uses shadows and tonal differences (surface tint) to show which elements are "above" others.
*   Level 0 (flat) to Level 5 (highest).

## Interaction
*   **Ripples**: Visual feedback on touch/click.
*   **Transitions**: Elements morph and expand smoothly (Shared Axis, Fade Through, Elevation).

## Key Components
*   **FAB (Floating Action Button)**: Primary action on a screen.
*   **Cards**: Contain content and actions about a single subject.
*   **Navigation Bar**: Bottom navigation for 3-5 top-level destinations.
*   **Top App Bar**: Contextual actions, navigation drawer access, and page title.
